    than a one-shot vector search baseline and still maintain acceptable turns to convergence and cognitive load?
\end{frame}

\begin{frame}{Evaluation Metrics}
  \begin{itemize}
    \item \textbf{Turns to convergence}: oracle turns from start to acceptance signal
    \item \textbf{Cognitive load per turn}: titles shown + clusters shown + question complexity, averaged per session
    \item \textbf{Oracle satisfaction rate}: fraction of sessions converging before the turn budget is exhausted
    \item \textbf{Drift detection rate}: fraction of injected contradictions correctly surfaced
    \item \textbf{LLM-as-Judge scores}: clustering coherence, question quality, preference profile fidelity
    \item All quantitative claims carry \textbf{95\% bootstrap confidence intervals}
  \end{itemize}
\end{frame}

\begin{frame}{Oracle Satisfaction}
  No external ground truth exists, Three complementary signals are used:
  \begin{itemize}
    \item \textbf{Explicit acceptance}: oracle says \textit{"perfect"} / \textit{"yes, that's it"}, or gives no corrective feedback for 2 consecutive turns
    \item \textbf{Pairwise probe}: after convergence, sample pairs from the accepted cluster and ask \textit{"do these two belong together?"} — validates internal coherence without any external label
    \item \textbf{Hidden-spec overlap} (simulated oracles only): each persona has a pre-written preference spec (genres, styles, exclusions); titles in the accepted cluster are scored against it — metric: fraction of titles satisfying all positive and no negative criteria
  \end{itemize}
\end{frame}
